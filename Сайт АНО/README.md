# Сайт АНО «Академия Молодых Лидеров»

Многостраничный сайт для автономной некоммерческой организации «Академия Молодых Лидеров».

## Структура

- **index.html** — главная страница
- **about.html** — Об АНО (миссия, руководство, документы)
- **directions/** — 8 направлений экосистемы
- **projects.html** — проекты и мероприятия
- **partners/** — раздел для партнёров (5 подразделов)
- **students.html** — студентам (анкета, программы, карьера)
- **media.html** — журнал «Мой Нижний»
- **contacts.html** — контакты, форма обратной связи
- **privacy.html** — политика конфиденциальности

## Дизайн

- **Основной цвет:** #1E3A5F
- **Дополнительный:** #4A90E2
- **Акцентный:** #F5A623
- **Фон:** #F5F7FA
- **Шрифты:** Montserrat (заголовки), Roboto (текст)
- **Иконки:** Material Symbols

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер (например, `npx serve`).

## Что нужно настроить

1. **Документы** — добавить PDF в папку `documents/`: устав, свидетельство, отчёты
2. **Реквизиты** — заменить в about.html и contacts.html на актуальные
3. **Соцсети** — добавить ссылки на VK, Telegram
4. **Карта** — подключить Яндекс.Карты или Google Maps в contacts.html
5. **Формы** — настроить обработку (backend или сервис типа Formspree)
